#Postman

Postman Collection of our KudosHub Email Validation Service API

Postman is an API Development Enviroment (ADE)

To download Postman, please visit https://www.getpostman.com/

Once you install postman, please import the file KudosHub_Email_Validation_ API_postman_collection.json

To get started testing and using our API, you will need to create an account on www.kudoshub.com to access your Email Validation API Key.
